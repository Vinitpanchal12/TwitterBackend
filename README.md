Twitter backend project using expressjs and mongodb.
Features includes:
              user will be able to create a post 
               	- image upload 
              	- tweet cannot be more than 250 char
              post visible to only who follows you
              comment on  a post  / tweet
              comment on a comment 
              like comment 
              user authentication
              pagination
              user profile
              	-name
              	-followers
              	-bio
              	-last tweets
